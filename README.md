# SMILES: Robust Self-supervised Multi-instance Learning with Structure Awareness


## Prerequisites

SMILES needs the following packages to be installed beforehand:

```
Python 3.8+

PyTorch 1.9+

PyTorch-Geometric 1.7 

Scikit-learn 0.24+

scipy

Numpy

tqdm

NetworkX
```

## Running Code

``` python Smiles.py ```
